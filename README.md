# Expediate 

_Expediate_ is a simple React Native app that helps users track their [Diet Quality Score (DQS)](https://www.triathlete.com/nutrition/race-fueling/do-you-know-your-diet-quality-score/).

## Get started

`npm i`

`npm start`

`npm run json-server`
